# 深圳 人才引进政策速查

- 状态：已重写为“满足什么条件 → 可以获得什么优势”结构
- 优先来源：广东政务服务网、深圳市人民政府、深圳市人社局等官方公开信息

## 一句话判断
深圳的人才引进现在很适合拆成两件事看：**先看能不能通过在职人才引进拿到深圳户口**，再看你是否属于**可领补贴的人才层级**。市级最明确、最稳定的补贴是**新引进博士人才生活补贴10万元**；本科、硕士补贴更多是**各区各自制定**，不宜写成全市统一标准。

---

## 1. 满足什么条件 → 可以获得什么优势

### 1.1 如果你符合在职人才引进条件

**你需要大致满足：**
- 身体健康；
- 已在深圳依法缴纳社会保险（符合豁免情形的除外）；
- 未参加国家禁止的组织及活动，无刑事犯罪记录。
- 并符合以下任一类型：
  1. **学历型人才**
     - 普通高等教育**本科以上学历**，**45周岁以下**；
     - 普通高等教育**专科以上学历**，**35周岁以下**。
  2. **专业技术资格人才**
     - 高级专业技术资格：**50周岁以下**；
     - 中级专业技术资格：**45周岁以下**；
     - 一般还需具有**中专以上学历**。
  3. **技能人才**
     - 高级技师：**45周岁以下**；
     - 技师：**40周岁以下**；
     - 高级技能职业资格：**35周岁以下**，且需在深圳参加社保满**3年以上**；
     - 职业资格还需符合深圳紧缺职业目录。
  4. **高层次人才 / 获奖人才 / 积分人才**
     - 经深圳认定的高层次人才；
     - 技能竞赛获奖者、技术能手、政府表彰人才；
     - 或综合评价达到**100分**，年龄**45周岁以下**。

**你可以获得：**
- 通过**在职人才引进**办理深圳户籍；
- 审批通过后，系统推送公安等部门，继续办理入户手续；
- 办理节奏相对清晰，且支持线上办理。

**但要注意：**
- 深圳的“人才引进”与“应届毕业生接收”不是同一条通道；
- 不同类型对年龄、社保、证书的要求差异很大，不能混写。

---

### 1.2 如果你是新引进博士人才

**你需要大致满足：**
- 引进时间或首次在深就业时间在**2021年9月1日及以后**；
- **35岁（含）以下**；
- 具有**博士研究生学历（学位）**。

#### 国内新引进博士人才
- 已取得**深圳户籍**；
- 首次申请时在深圳**全职工作**；
- 引进后通过深圳工作单位缴纳社保累计**3个月以上**（不含补缴）；
- 续发申请时，仍具有深圳户籍、仍在深圳全职工作，且自首发申请起**连续缴纳社保满12个月**。

#### 海外新引进博士人才
- 首次在深圳就业的港澳台或外籍博士；
- 首次申请时在深圳**全职工作**，并累计缴纳社保**3个月以上**；
- 续发申请时继续全职工作，且自首发申请起**连续缴纳社保满12个月**；
- **不要求深圳户籍**。

**你可以获得：**
- **市级新引进博士人才生活补贴 10万元/人**；
- 发放方式为：
  - 首发**30%**；
  - 续发**70%**。

**这条政策的特点是：**
- 深圳市级对博士支持非常明确；
- 对海外博士也比较友好，因为不要求深圳户籍。

---

### 1.3 如果你是本科 / 硕士人才

**你需要知道：**
- 深圳市级公开口径目前并未统一公布本科、硕士的固定补贴标准；
- **本、硕人才入户补贴由各区制定和实施**。

**你可能获得：**
- 在满足区级条件后，领取所在区的**新引进优秀青年人才入户补贴**；
- 常见会关联：
  - 学历层级；
  - 年龄；
  - 是否取得深圳户籍；
  - 是否在本区就业；
  - 社保连续缴纳月数等。

**但要注意：**
- 因为区与区差异较大，本次不宜在市级文件里写死具体金额；
- 应明确标注：**待按区复核**。

---

### 1.4 如果你是高层次、专业技术、技能或积分型人才

**你需要大致满足：**
- 属于深圳认定的高层次人才；
- 或有高级/中级专业技术资格；
- 或属于高级技师、技师、高级技能人才；
- 或通过综合评价达到积分入户标准。

**你可以获得：**
- 通过不同类别的**人才引进入户**通道进入深圳；
- 对很多非博士人才来说，真正最有价值的市级支持仍然是**先拿到深圳户口**。

**但要注意：**
- 深圳市级统一的购房补贴、租房补贴、创业补贴口径并不清晰；
- 这部分更多落在各区、各类专项人才计划中。

## 2. 速查表

| 条件维度 | 深圳人才引进口径 |
|---|---|
| 学历 | 大专35岁以下、本科45岁以下、博士另有专项补贴政策 |
| 年龄 | 35/40/45/50 岁分档明显，随通道不同而变 |
| 社保 | 在职人才一般需已在深缴社保；高级技能类明确需满3年；博士补贴首发需3个月、续发需连续12个月 |
| 户籍 | 人才引进本身就是落户通道；国内博士补贴首发需深圳户籍，海外博士可不要求 |
| 职称/技能 | 中高级职称、高级工/技师/高级技师、获奖人才都能成为通道 |
| 留学 | 留学回国人员可走在职人才引进；海外博士可申领市级生活补贴 |
| 核心优势 | **落户深圳** + **博士10万元市级生活补贴** |
| 补贴特点 | 博士看市级，本硕看区级 |

---

## 3. 当前更适合怎么理解深圳
- 如果你是博士，深圳的市级支持最明确；
- 如果你是本科或硕士，深圳并不是“全市统一发钱”，而是要具体看你落在哪个区；
- 如果你最在意的是**入户效率和年轻人才通道**，深圳依然非常有吸引力；
- 如果你想要全国统一口径式的购房、租房补贴，深圳更适合按区拆开看。

  - 购房、租房、创业支持是否存在适用于特定区或特定人才层级的专项政策。
