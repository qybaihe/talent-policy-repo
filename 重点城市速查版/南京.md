# 南京 人才引进政策速查

- 状态：已精修，补充政策有效期与住房判断规则
- 优先来源：南京市人社局等官方公开信息

## 一句话判断
南京当前最容易核实、最有辨识度的人才优势，不是单一落户，而是**住房租赁补贴 + 重点产业提标 + 海外人才创新创业支持**。落户便利依然存在，但本轮能稳定核实到的最新正式口径，主要来自人社系统，而不是公安落户办事指南，因此在速查里更适合把南京理解成**“安居和产业支持较强”的城市**。

---

## 1. 满足什么条件 → 可以获得什么优势

### 1.1 如果你是普通高校毕业生，在南京就业创业

**你需要大致满足：**
- 属于博士、硕士、学士，或符合条件的大专学历毕业生；
- 在南京就业或创业；
- 与住房租赁补贴相关的核心条件通常包括：
  - 在宁连续参保；
  - 申请当月在宁**无房**；
  - 具体社保起算、断缴等规则按申报系统和实施细则执行。
- 2026年优化文件明确：新增人员连续参保首月自**2026年1月1日**起算，最早可于**2026年7月1日**申领。

**你可以获得：**
- 申请**高校毕业生住房租赁补贴**；
- 补贴累计发放**不超过36个月**；
- 这对刚到南京发展的青年人才，是相对持续、可预期的支持。

---

### 1.2 如果你是大专毕业生，且具备技能证书

**你需要大致满足：**
- 具有全日制大专学历；
- 同时具有**高级工及以上职业资格证书或职业技能等级证书**；
- 在南京就业并符合连续参保、无房等条件。

**你可以获得：**
- **500元/月**住房租赁补贴；
- 若在南京人工智能（软件）、机器人、生物医药、新一代信息通信等重点领域规上企业就业，可提高到**600元/月**。

**这条政策的含义是：**
- 南京对技能型青年人才也给了明确安居支持；
- 不只是本科、硕士、博士受益。

---

### 1.3 如果你是硕士 / 博士，并进入重点产业企业

**你需要大致满足：**
- 在南京人工智能（软件）、机器人、生物医药、新一代信息通信等重点领域**规上企业**就业；
- 符合学历、连续参保、在宁无房等条件。

**你可以获得：**
- 重点领域规上企业新引进毕业生租房补贴提标：
  - 博士：**2400元/月**；
  - 硕士：**960元/月**。

**这意味着：**
- 南京会把人才支持和产业导向绑得比较紧；
- 如果你恰好进入重点赛道，待遇会比普通口径更好。

---

### 1.4 如果你是重点大学学士及以上，并进入人工智能（软件）企业

**你需要大致满足：**
- 进入南京人工智能（软件）企业工作；
- 属于重点大学学士及以上毕业生；
- 符合社保、无房等安居申报条件。

**你可以获得：**
- 博士：**2400元/月**；
- 硕士：**960元/月**；
- 学士：**720元/月**。

**这条政策的特点是：**
- 南京对软件和人工智能赛道有额外倾斜；
- 本科生在这个口径下也能拿到相对明确的租房支持。

---

### 1.5 如果你是世界前200高校博士 / 医学类前200高校博士

**你需要大致满足：**
- 属于世界前200强高校博士；
- 并到南京高校院所、国防科工单位工作；
- 或属于世界前200强医学类院校博士，并到南京二级及以上医疗机构工作。

**你可以获得：**
- **2000元/月**住房租赁补贴。

**这条政策说明：**
- 南京对高层次、国际化人才也有明确倾斜；
- 尤其科研、国防、医疗方向更明显。

---

### 1.6 如果你是海外留学人才，想在南京创新创业

**你需要大致满足：**
- 在国（境）外学习并取得学士及以上学位；
- 或在国内取得本科及以上学历（或中级及以上职称）后，到国（境）外进修、访学或从事博士后工作**1年以上**；
- 来宁开展创新创业或技术研发活动。

**你可以获得：**
- **留学人员科技创新项目资助**：
  - A类：**10万元**；
  - B类：**5万元**；
  - C类：**3万元**。
- 通过市、区两级企业专家工作室引进海外高层次人才开展研发攻关的，给予**最高100万元项目资助**。
- 还可获得专家咨询、创投对接、宣传推介、疗休养等配套支持。

**这意味着：**
- 南京对留学人才的核心优势，不只是落户，而是**创新创业资金 + 项目支持**。

## 2. 速查表

| 条件维度 | 南京人才引进口径 |
|---|---|
| 学历 | 大专、本科、硕士、博士均可纳入不同层级支持 |
| 年龄 | 本轮未稳定抓到最新落户正式年龄细则，暂标待复核 |
| 社保 | 住房租赁补贴与连续参保直接相关 |
| 户籍 | 本轮已核实政策未突出“必须南京户籍”才能申请租房补贴 |
| 技能 | 大专+高级工及以上证书可纳入补贴 |
| 留学 | 留学人才创新创业项目资助较明确 |
| 核心优势 | **租房补贴持续、重点产业提标、留学人才项目资助** |
| 补贴特点 | 更偏“安居支持+产业导向”，而非简单一次性安家费 |

---

## 3. 当前更适合怎么理解南京
- 如果你看重**连续性的住房租赁补贴**，南京很值得关注；
- 如果你进入重点产业、重点企业，支持力度会明显提高；
- 如果你是海外留学人才，南京在**项目资助**方面比较有辨识度；
- 但如果要写成“落户年龄一览表”，本轮还需要再补公安口径，当前宜保守标注**待复核**。
