# 南京 人才引进政策速查

- 状态：已精修，补充公安落户办理框架、学历落户材料口径边界与政策有效期
- 优先来源：南京市人社局等官方公开信息；公安落户部分本轮仅保守补到办理框架，细项维持待复核

## 一句话判断
南京当前最容易核实、最有辨识度的人才优势，不只是**住房租赁补贴 + 重点产业提标 + 海外人才创新创业支持**，而且落户办理上大概率仍延续“公安户政受理、线上线下可办”的便利框架。只是本轮在当前环境下，南京公安与政务站点存在较强 403 拦截，**学历落户年龄上限、最新材料清单、精确办事指南页面**未能直接抓到原文，因此正文只写入能稳妥落笔的办理框架，避免把未坐实信息写死。

## 1. 满足什么条件 → 可以获得什么优势

> 说明（本轮公安落户补证情况）：
> - 已能稳定打开南京市公安局官网首页，但用于“户政/办事指南/户口迁入（含学历/人才落户）”的具体事项页与江苏政务服务网事项页，在本运行环境下仍存在“无法访问/拦截/需要交互验证”等情况，导致**年龄上限、逐项材料清单、线上入口URL**无法用可引用原文坐实。
> - 因此：本文件对“学历/人才落户”仅写入**可被官方框架支撑的结论**（主管部门与办理形态），并把待复核点进一步收敛到“需要补到的官方事项页”。

### 1.1 如果你是普通高校毕业生，在南京就业创业

**你需要大致满足：**
- 属于博士、硕士、学士，或符合条件的大专学历毕业生；
- 在南京就业或创业；
- 与住房租赁补贴相关的核心条件通常包括：
  - 在宁连续参保；
  - 申请当月在宁**无房**；
  - 具体社保起算、断缴等规则按申报系统和实施细则执行。
- 2026年优化文件明确：新增人员连续参保首月自**2026年1月1日**起算，最早可于**2026年7月1日**申领。

**你可以获得：**
- 申请**高校毕业生住房租赁补贴**；
- 补贴累计发放**不超过36个月**；
- 这对刚到南京发展的青年人才，是相对持续、可预期的支持。

---

### 1.2 如果你是大专毕业生，且具备技能证书

**你需要大致满足：**
- 具有全日制大专学历；
- 同时具有**高级工及以上职业资格证书或职业技能等级证书**；
- 在南京就业并符合连续参保、无房等条件。

**你可以获得：**
- **500元/月**住房租赁补贴；
- 若在南京人工智能（软件）、机器人、生物医药、新一代信息通信等重点领域规上企业就业，可提高到**600元/月**。

**这条政策的含义是：**
- 南京对技能型青年人才也给了明确安居支持；
- 不只是本科、硕士、博士受益。

---

### 1.3 如果你是硕士 / 博士，并进入重点产业企业

**你需要大致满足：**
- 在南京人工智能（软件）、机器人、生物医药、新一代信息通信等重点领域**规上企业**就业；
- 符合学历、连续参保、在宁无房等条件。

**你可以获得：**
- 重点领域规上企业新引进毕业生租房补贴提标：
  - 博士：**2400元/月**；
  - 硕士：**960元/月**。

**这意味着：**
- 南京会把人才支持和产业导向绑得比较紧；
- 如果你恰好进入重点赛道，待遇会比普通口径更好。

---

### 1.4 如果你是重点大学学士及以上，并进入人工智能（软件）企业

**你需要大致满足：**
- 进入南京人工智能（软件）企业工作；
- 属于重点大学学士及以上毕业生；
- 符合社保、无房等安居申报条件。

**你可以获得：**
- 博士：**2400元/月**；
- 硕士：**960元/月**；
- 学士：**720元/月**。

**这条政策的特点是：**
- 南京对软件和人工智能赛道有额外倾斜；
- 本科生在这个口径下也能拿到相对明确的租房支持。

---

### 1.5 如果你是世界前200高校博士 / 医学类前200高校博士

**你需要大致满足：**
- 属于世界前200强高校博士；
- 并到南京高校院所、国防科工单位工作；
- 或属于世界前200强医学类院校博士，并到南京二级及以上医疗机构工作。

**你可以获得：**
- **2000元/月**住房租赁补贴。

**这条政策说明：**
- 南京对高层次、国际化人才也有明确倾斜；
- 尤其科研、国防、医疗方向更明显。

---

### 1.6 如果你是海外留学人才，想在南京创新创业

**你需要大致满足：**
- 在国（境）外学习并取得学士及以上学位；
- 或在国内取得本科及以上学历（或中级及以上职称）后，到国（境）外进修、访学或从事博士后工作**1年以上**；
- 来宁开展创新创业或技术研发活动。

**你可以获得：**
- **留学人员科技创新项目资助**：
  - A类：**10万元**；
  - B类：**5万元**；
  - C类：**3万元**。
- 通过市、区两级企业专家工作室引进海外高层次人才开展研发攻关的，给予**最高100万元项目资助**。
- 还可获得专家咨询、创投对接、宣传推介、疗休养等配套支持。

**这意味着：**
- 南京对留学人才的核心优势，不只是落户，而是**创新创业资金 + 项目支持**。

---

### 1.7 如果你更关心“南京学历落户/人才落户怎么办”（公安口径待补证）

**本轮能稳妥写入的部分（已坐实到“主管单位”层级）：**
- 该类“户口迁入/落户”事项的主管受理口径，原则上属于**南京市公安局户政**职责范围（公安机关负责户籍登记管理）。
  - 引用入口（官网主站，可作为主管单位引用）：南京市公安局 https://gaj.nanjing.gov.cn/
- 办理形态上，按全国公安户政事项的通行做法，通常会存在“线上申报/预约预审 + 线下窗口受理”的组合；但**是否全程网办、是否必须本人到场、能否跨区通办**等，必须以南京公安或江苏政务服务网的“事项办事指南”原文为准。

**本轮无法坐实、但可先给出“材料框架（非南京最新版清单）”的部分：**
- 学历/人才落户常见材料结构通常围绕：
  - 申请人身份材料（身份证等）；
  - 原户籍/迁移信息材料；
  - 学历学位证明与认证材料（如学信网相关证明/国境外学历认证等）；
  - 落户地址材料（自有房产/亲属房产/单位集体户/人才集体户等对应证明）；
  - 就业劳动关系、社保缴纳等材料（**如该通道要求**）。

**待复核（必须补到“官方办事指南”才能写死的关键卡点）：**
1) 年龄上限（如是否存在“本科/硕士/博士不同年龄线”）；
2) 材料清单逐项口径（含是否要求社保/劳动合同/调档/无犯罪记录等）；
3) 办理入口（江苏政务服务网事项URL或南京公安线上系统入口）、办理时限、到场次数；
4) 是否区分“学历落户/人才引进落户/就业落户”等不同事项名。

> 注：本运行环境中，江苏政务服务网主站可访问，但部分接口/事项页出现“无法访问”页面，暂无法拉取到可引用的事项办事指南原文。

## 2. 速查表

| 条件维度 | 南京人才引进口径 |
|---|---|
| 学历 | 大专、本科、硕士、博士均可纳入不同层级支持；学历落户具体分层条件仍待公安口径坐实 |
| 年龄 | 住房补贴部分本轮不以年龄为主限制；**学历落户年龄上限仍待复核** |
| 社保 | 住房租赁补贴与连续参保直接相关；落户是否要求就业/社保按公安办事指南确认 |
| 户籍 | 租房补贴本轮未见必须先有南京户籍；落户办理本身属户籍迁入事项 |
| 技能 | 大专+高级工及以上证书可纳入补贴 |
| 留学 | 留学人才创新创业项目资助较明确；留学学历落户材料细项待公安页补证 |
| 办理方式 | **可保守理解为公安户政线上+线下办理框架存在，精确到场次数/是否全程网办待复核** |
| 材料清单 | 当前仅能保守写到身份证、户籍、学历认证、落户地址/集体户、就业社保等常见框架；**最新版南京清单待复核** |
| 核心优势 | **租房补贴持续、重点产业提标、留学人才项目资助** |
| 补贴特点 | 更偏“安居支持+产业导向”，而非简单一次性安家费 |

## 3. 当前更适合怎么理解南京
- 如果你看重**连续性的住房租赁补贴**，南京很值得关注；
- 如果你进入重点产业、重点企业，支持力度会明显提高；
- 如果你是海外留学人才，南京在**项目资助**方面比较有辨识度；
- 如果你关心“能不能落户、怎么落户”，南京应理解为：**落户路径大概率成熟，但本轮尚缺公安正式细则页面的原文抓取**；
- 所以南京现在最稳妥的表述，不是“落户条件完全清楚”，而是**安居端已坐实较多、公安落户端已缩小待复核范围**。

## 4. 如能继续补证，下一步最值得抓什么
1. **南京公安局户政办事指南原文页**：优先确认学历落户标题、受理单位、办理时限、到场次数、是否可网办；
2. **江苏政务服务网对应事项页**：优先确认材料清单、表单下载、办理深度、咨询电话；
3. **南京 12345 / 公安问答类官方答复**：优先确认学历落户年龄上限、本科硕士博士是否分档；
4. **区级官网 / 房产或人才专窗**：如有区级人才房票、购房支持，再决定是否增补独立小节。
