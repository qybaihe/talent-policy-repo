# 珠海 人才引进政策速查

- 状态：已四次精修；本轮重点补到 **香洲区“一免两减”实施细则正文**，把香洲区的学历、年龄、首次来珠时间、社保/个税、无房等条件从“待复核”升级为“已坐实”，并继续稳住高新区/金湾区的执行边界
- 优先来源：珠海市人民政府门户网站、各区政府官网、广东政务服务网

## 一句话判断
珠海当前最值得关注的青年人才住房支持，已经不是旧版现金补贴，而是 **“新来珠青年人才住房‘一免两减’”**。截至目前，**市级方向和核心优惠内容已能由官方公开口径坐实**；**香洲区实施细则已能坐实到正式通知层级**；**高新区、金湾区已能坐实到具体批次配租/资格审核公告层级**。但如果要把 **学历门槛、首次来珠时间、年龄、社保、是否无房** 写成现行统一标准，仍需要继续抓正式实施细则原文逐项核对。

---

## 1. 先看市级：珠海已经明确推动青年人才住房“一免两减”

### 1.1 已坐实的市级核心口径

根据珠海市近年官方公开报道、政策汇编页面及政府工作报告口径，珠海对**新来珠青年人才**的人才住房支持，核心方向已经比较清楚：

- 政策名称方向已明确为：**新来珠青年人才住房“一免两减”**；
- 优惠结构已能由官方公开口径坐实为：
  - **首年免租金**；
  - **第二年按同区域同类型同品质市场商品住房租金的30%收取**；
  - **第三年按50%收取**；
- 珠海 2026 年政府工作报告已继续把青年人才安居保障写入公开表述，说明该方向并非一次性宣传口径，而是在持续推进；
- 市级公开报道已把该政策描述为珠海吸引青年人才的重要“礼包”之一。

### 1.2 目前还不能直接写死的项目

虽然“一免两减”的优惠结构已较稳，但以下条件目前仍不宜在总表里写成“珠海全市统一现行标准”：

- **适用学历门槛**：是否统一要求本科及以上，或是否细分至大专/技师/硕士等；
- **首次来珠时间**：是否统一要求某一特定日期后首次来珠就业创业；
- **年龄限制**：是否按学历层级分别设置；
- **社保要求**：是否要求在珠连续缴纳社保满一定月数；
- **住房条件**：是否要求本人及家庭在珠无自有住房、未享受其他住房保障；
- **申请入口**：是否市级统一平台受理，还是由各区各自发布房源并审核。

也就是说：**珠海市级的“一免两减”方向、优惠结构已坐实；但统一适用条件还没有完全坐实。**

## 2. 现行珠海，更稳的是“引进通道 + 安居支持”双线并看

### 2.1 如果你走珠海市“直接核准引进（含高技能人才引进）”

广东政务服务网已能稳定坐实以下事项：

- 事项名称：**直接核准引进（含高技能人才引进）（市外人才引进）**；
- 主管部门：珠海市人力资源和社会保障系统；
- 办理特点：
  - **承诺办结时限 8 个工作日**；
  - **法定办结时限 15 个工作日**；
  - **到现场次数 0 次**；
  - 支持**网上办理、窗口办理、快递申请**；
  - 支持**跨省通办**，并对**港澳台地区支持跨境通办**。

这条线的意义是：

- 珠海的人才引进入口是清晰可办的；
- 对市外来珠人才、港澳台人才尤其友好；
- 但其**具体学历/年龄/社保条件**，仍需继续抓受理条件原文，不宜在这里代替正式指南自行归纳。

### 2.2 如果你最关心安居成本

那珠海当前最强的吸引点不是旧版一次性补贴，而是：

- **新来珠青年人才住房“一免两减”**；
- 部分区域还叠加了：
  - **区级人才住房租赁管理细则**；
  - **面向用人单位的六折配租**；
  - **按批次房源配租 / 资格审核 / 抽签公示**。

这说明珠海现在的安居支持，已经逐步从“发一笔钱”转成“给房源、降租金、做持续性保障”。

## 3. 区级补充：香洲 / 高新区 / 金湾已出现明确执行分化

### 3.1 香洲区：已坐实到实施细则正文层级，可直接落到准入条件

这是本轮最关键的新增进展。

通过香洲区住建局官网公开页面及其附件 PDF，已抓到**《关于印发〈香洲区关于试行新来珠青年人才住房“一免两减”政策措施的实施细则〉的通知》**正文，可直接坐实以下条件：

- **适用对象范围**：
  - 符合香洲区主导产业发展定位；
  - **工商注册及税务征管关系在香洲区的企业**；
  - 从区外全职新引进的青年人才；
  - 满足以下其一：
    1. **普通高等教育全日制本科（含）以上学历**；
    2. **中级以上职称**；
    3. **技师以上国家职业资格（职业技能等级）**；
    4. **港澳永久性居民、获批香港/澳门有关人才计划的港澳居民、台湾地区居民**；
    5. **在珠海首次申领 A 类、B 类外国人来华工作许可证的外籍人才**。
- **年龄要求**：
  - 全日制博士、高级职称、港澳台人才、外籍人才：**40 周岁以下**；
  - 全日制硕士、中级职称、高级技师：**35 周岁以下**；
  - 全日制本科、技师：**30 周岁以下**。
- **首次来珠 / 新引进时间要求**：
  - **自 2023 年 6 月 1 日后首次引进到香洲区就业**；
  - 以到香洲区工作后**首次缴纳社保（或工资薪金个税）**及**首次签订劳动合同**中较早一项为准；
  - **首次引进 1 年内须提出申请**；
  - 对 **2023-06-01 至 2024-08-05** 引进的，申请时间**不得超过 2025-08-05**。
- **劳动合同 / 社保个税要求**：
  - 须由香洲区企业**全职引进**；
  - 与用人单位签订**1 年以上劳动合同**；
  - 须依法在香洲区**缴纳社会保险费或工资薪金个人所得税**。
- **无房与未享受其他保障要求**：
  - 申请人**未在珠海市享受过货币、实物住房等住房保障措施**；
  - 申请人及其**配偶、未成年子女**在珠海**无任何形式不动产权份额**；
  - **近 5 年内**在珠海**无不动产转移登记记录**；
  - 未因违反住房保障和人才政策被行政处罚或纳入不良行为记录。
- **保障标准**：
  - A 类（博士/高级职称）：原则上成套住房，**不超过 100㎡**；
  - B 类（硕士/中级职称/高级技师）：单身公寓或合租，**不超过 50㎡**；
  - C 类（本科/技师）：原则上**一人一间**，**不超过 30㎡**；
  - 港澳台人才、外籍人才：**按不低于 B 类标准保障**；
  - 配租面积可**上浮 20%**，且**无需补齐差价**；超出上浮 20% 的部分按市场价补租。

同站还可见多条配套问答，以及 **《珠海市香洲区人才住房租赁管理实施细则（2025年修订）》**。

**这意味着：**

- 香洲区已经不只是“有通知标题”，而是已能直接落到**学历、年龄、首次来珠时间、社保/个税、无房、未享受其他住房保障**等关键准入条件；
- 目前珠海区级里，香洲是最适合拿来做“已坐实样本”的一块；
- 但上述条件是**香洲区实施细则口径**，仍不应自动外推为“珠海全市统一标准”。

珠海 2019 年那套“新引进专业技术人才、高技能人才、青年人才住房（租房和生活）补贴”问答，历史上很清楚，但页面已标示相关文件**已失效**。

旧口径大致包括：

- 全日制本科、技师：**2.6 万元**；
- 全日制硕士、高级技师、中级职称：**3.8 万元**；
- 副高、正高：对应更高住房补贴；
- 通常涉及：本市户籍、劳动合同年限、连续社保和个税等要求。

这部分今天还能用来做两件事：

1. 证明珠海曾经有过比较明确的青年人才现金补贴分档；
2. 对比说明珠海现在的政策重心正转向**人才住房保障**。

但它**不能再直接写成现行补贴标准**。

## 6. 当前更适合怎么理解珠海

如果从“应届生 / 青年人才去不去”这个角度看，珠海的现实吸引力主要来自三件事：

1. **人才引进入口清晰**：直接核准引进事项稳定、可网办、通办能力强；
2. **安居支持强于现金补贴叙事**：一免两减对新来珠青年更实用；
3. **区级开始形成分层执行**：香洲偏细则化，高新区偏批次化执行，金湾还叠加用人单位六折配租。

所以，现阶段对珠海最稳妥的结论不是“本科补多少、硕士补多少”，而是：

> **珠海现行青年人才住房支持已进入“一免两减”阶段，市级方向明确，香洲/高新区/金湾已有区级执行落地，但学历、首次来珠时间、年龄、社保等统一门槛仍需继续补正式细则。**

- 核心优惠结构已可坐实：**首年免租、第二年30%、第三年50%**；
- 珠海人才引进事项已由广东政务服务网坐实：**直接核准引进（含高技能人才引进）**可在线办理，承诺 8 个工作日、法定 15 个工作日、到场 0 次；
- 香洲区已坐实到**“一免两减”实施细则正文**层级，已可直接确认学历、年龄、首次来珠时间、劳动合同、社保/个税、无房及未享受其他住房保障等条件；
- 高新区已坐实到**配租公告、资格审核、名单公示、违规整治通知**层级；
- 金湾区已坐实到**“一免两减”抽签结果公示**及**用人单位六折配租公示**层级；
- 2019 年旧版青年人才现金补贴体系**确已失效**，不能直接作为现行标准引用。

### 9.2 待复核
- “一免两减”**市级正式实施文件**的稳定原文链接；
- 现行适用对象在**市级层面**是否统一要求：**本科及以上 / 指定学历层次 / 技能人才纳入**；
- 是否统一要求某一时间点后**首次来珠就业创业**（香洲区已坐实为 2023-06-01 后首次引进到区内就业）；
- 是否存在全市统一的**年龄上限**（香洲区已分档坐实）；
- 是否存在全市统一的**社保连续缴纳要求**（香洲区已坐实为在区内缴纳社保或工资薪金个税，但“连续满 X 月”尚未见统一市级口径）；
- 是否统一要求**本人及家庭在珠无房**、未享受其他住房保障（香洲区已坐实）；
- 横琴合作区是否存在独立但正在执行的青年人才住房细则。
