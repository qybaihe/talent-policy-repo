## 本轮“待复核清除专项”重点
- 已优先处理：北京、上海、广州、深圳、杭州、南京、苏州、成都、武汉、合肥、珠海。
- 重点核实项：年龄上限、社保月数、学历层级门槛、补贴金额、市级统一 or 区级差异、政策有效期/续期。
- 使用原则：优先政府官网、政务公开、办事指南；不能完全坐实的继续保留“待复核”，但缩小不确定范围。
