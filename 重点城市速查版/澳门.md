# 澳门 人才引进与居留政策速查

- 状态：首轮结构化完成；已按澳门口径改写为“人才引进制度 + 居留资格 + 横琴联动 + 生活成本/税务/就业环境”结构，不套用内地落户模板
- 优先来源：澳门特区政府入口网站、澳门《公报》/法律法规公开页、治安警察局出入境与居留公开服务页、横琴粤澳深度合作区官方公开页

## 一句话判断
澳门更适合 **已经具备较强学历、专业资历、行业影响力或国际化履历，且愿意把职业发展落到文旅会展、中医药大健康、现代金融、高新技术、葡语国家相关业务等方向的人**。如果你主要想要一个像内地城市那样“学历够了就能直接落户、再领一笔通用补贴”的结果，澳门通常**不是**最好选项；它更像是 **“先看你能否进入人才引进制度，再看是否能转化为居留资格与长期发展空间”**。

## 1. 主要人才引进制度：高端人才、优秀人才、高级专业人才怎么理解

### 1.1 澳门当前的人才引进主线，不是“落户”，而是法定人才引进制度

澳门这几年的人才政策，核心不是内地常见的“人才落户 + 社保月数 + 现金补贴”，而是围绕 **《人才引进法律制度》** 搭建的法定引进框架。

目前最值得先记住的判断是：

- 澳门已经建立了专门的人才引进法律制度；
- **法源已经可以坐实**：第 **7/2023号法律《人才引进法律制度》** 于 **2023年7月1日** 生效，配套的第 **19/2023号行政法规《人才引进法律制度施行细则》** 与第 **20/2023号行政法规《人才发展委员会》** 亦于同日生效；
- 制度按人才层级区分为 **高端人才、优秀人才、高级专业人才**；
- 人才引进与澳门整体经济适度多元发展方向绑定，不是泛化地“什么专业都大量要”；
- 对申请人而言，最关键的问题不是“能不能直接拿澳门户口”，而是：
  1. 你属于哪一类人才；
  2. 你能否进入审批名单；
  3. 获批后对应的是怎样的居留资格、续期要求和在澳实际发展空间。

### 1.1.1 本轮已打实的法律与制度骨架

依据澳门特区政府 2023-06-26 新闻稿《澳门特别行政区〈人才引进法律制度〉及配套行政法规7月1日起生效》、以及同日公布的《人才引进法律制度施行细则》草案说明，可先把制度骨架写实为：

- **第7/2023号法律《人才引进法律制度》**：主法律，2023年7月1日起生效；
- **第19/2023号行政法规《人才引进法律制度施行细则》**：配套实施细则，进一步规范各类人才引进计划的制定、公布、申请、审批、居留许可办理、有效期与续期程序、转计划程序、税务优惠申请审批等；
- **第20/2023号行政法规《人才发展委员会》**：配套设定人才发展委员会；
- 制度主管链条可坐实为：**人才发展委员会、人才引进评审委员会、治安警察局** 三方参与，最终由 **行政长官** 对个案作决定。

这部分已经不再属于“待核实方向”，而是可以直接写入正文的法源级信息。

### 1.2 高端人才：更像“少而精”的定向引进

更稳妥的理解方式是：

- **高端人才** 通常对应在国际、国家或行业层面已有较高成就、影响力或稀缺性的人；
- 这类人群往往不是靠“毕业院校 + 年龄”去竞争，而是靠 **代表性成果、行业地位、科研/产业带动能力、重大平台价值** 来判断；
- 澳门设置这一层，说明它并不是只想吸纳普通就业人口，而是希望直接引入能对重点产业、重点机构、重点平台产生带动效应的人才。

更适合哪些人：
- 顶尖科研人才、头部学者；
- 有国际奖项、重大项目、头部企业高层经历的人；
- 能明显带动澳门重点产业或重大平台的人。

当前已经能写实到的官方口径：
- 按澳门特区政府 2023-08-04 公布的三类人才计划首轮申报信息，**高端人才计划** 旨在引进 **“具备卓越才能或技术能力，并取得国际公认杰出成就或在特定领域作出重大贡献的人士”**；
- 申请人须 **具备计划所订的高端人才认定标准所载的任一成就或职衔**；
- 首轮已公开的高端人才计划申请期为 **2023-08-04 至 2024-02-03**。

当前不能写死的地方：
- 高端人才认定标准所列全部成就/职衔目录，本稿本轮尚未直接抓到完整官方清单原文；
- 因此不宜自行总结成“必须满足 A/B/C 三条”。

### 1.3 优秀人才：更像“有明确竞争力、能补产业短板的人”

更稳妥的理解方式是：

- **优秀人才** 不是顶尖塔尖人才，但通常也不是普通应届毕业生层级；
- 更可能对应 **具备较强专业能力、行业经验、学历背景、创新创业能力或国际化履历**，并能补澳门重点产业与公共发展短板的人；
- 实操上，这一类通常会是制度里最值得多数中高端职业人关注的层级。

更适合哪些人：
- 具有较强履历的硕博人才；
- 现代金融、会展商贸、中医药大健康、高新技术、文化体育等方向的骨干人才；
- 有跨境业务、葡语国家业务、国际平台经验的人。

当前已经能写实到的官方口径：
- 2023年首轮公开接受申请时，已明确落地的 **优秀人才计划** 至少包括 **大健康产业优秀人才计划**；
- 官方表述为：旨在引进 **能推动大健康产业发展、具专业经验及技术能力，并在指定专业范畴表现卓著的人士**；
- 首轮列明的专业范畴包括：**医疗服务、医院和药厂管理、智慧医疗技术开发、创新药物研发及监督管理、医疗设备和器械标准化、医药和生物教研、医药注册/产品专利/市场活动**；
- 申请人须同时符合：
  1. **相关学科学士或以上学位**（医学、药学、公共卫生、医务行政管理、生命科学或相类学科）且具 **不少于4年** 相关专业经验；**或** 其他学科学士或以上学位且具 **不少于6年** 相关专业经验；
  2. **年满21岁**；
  3. **具良好中文、葡文或英文任一语文表达能力**；
  4. 按计划计分表 **300分中取得200分**。
- 上述大健康产业优秀人才计划首轮申请期为 **2023-08-04 至 2023-12-03**。

当前不能写死的地方：
- 现代金融、高新科技、文化体育等其他重点产业的优秀人才计划，后续虽有“陆续公布”官方口径，但本稿本轮未逐一抓到每个产业批次的稳定原文；
- 因此目前只把 **已坐实的大健康产业优秀人才计划** 条件写入正文，不外推到全部产业。

### 1.4 高级专业人才：更像“澳门产业与机构真实需要的专业骨干”

更稳妥的理解方式是：

- **高级专业人才** 往往更接近“具备明确专业资历、职业经验和岗位承接能力的人”；
- 这类人不一定是学术或产业塔尖，但在澳门现实产业结构里，可能是最容易形成“岗位—雇主—居留资格”闭环的一类；
- 如果申请人本身就在医药、工程、金融、信息技术、教育、专业服务等领域有成熟履历，这一层往往比“高端人才”更现实。

按已公开、可坐实的首轮计划口径，**大健康产业高级专业人才计划** 的正式申请条件至少包括：
- 申请人须在指定专业范畴具专业经验及技术能力，专业范畴与大健康产业优秀人才计划相同，包括 **医疗服务、医院和药厂管理、智慧医疗技术开发、创新药物研发及监督管理、医疗设备和器械标准化、医药和生物教研、医药注册/产品专利/市场活动**；
- 学历/经历门槛为：
  1. 具 **医学、药学、公共卫生、医务行政管理、生命科学或相类学科** 学士或以上学位，且具 **不少于2年** 相关工作经验；**或**
  2. 具其他学科学士或以上学位，且具 **不少于4年** 相关工作经验；
- **年满21岁**；
- **具良好中文、葡文或英文任一语文表达能力**；
- 按计划计分表 **300分中取得150分**；
- **已获本地雇主聘用，或承诺于居留许可决定生效之日起90日内聘用**；
- 拟担任的须是计划所列 **人力资源紧缺的专业职务**；
- 上述职务的 **年薪须达澳门元600,000元**，以全年税前总报酬计算。

适合怎样的人：
- 有专业执照、职称、资格证或中高年限从业经历的人；
- 能进入澳门重点企业、学校、医院、科研机构、专业服务机构的人；
- 不是泛泛求职，而是能和具体岗位需求匹配的人。

当前不能写死的地方：
- 本轮已坐实的是 **大健康产业高级专业人才计划** 的首轮公开条件；其他重点产业的高级专业人才计划细则，仍应继续按官方批次逐个补原文。

### 1.5 澳门这套制度的核心特点

和内地很多城市相比，澳门这套制度更像：

- **法定配额/计划管理 + 分类引进 + 与产业方向绑定**；
- 不宜理解成“只要学历够、年龄合适，就能稳定拿身份”；
- 更接近香港部分输入计划的逻辑，但澳门体量更小、岗位池更窄、产业更集中，因此 **选择面更小，但匹配度要求更高**。

## 2. 居留资格 / 续期逻辑：重点看什么

### 2.1 获批人才引进，不等于立刻取得“永久身份”

澳门这里最容易误解的一点是：

- 人才引进制度解决的是 **“是否获准按有关制度进入澳门并取得相应居留基础”**；
- 它并不等同于“像内地落户那样一次办完”；
- 对大多数申请人而言，更关键的是后续的 **居留许可、续期、维持条件、在澳实际停留与发展安排**。

更稳的写法是：
- 澳门应理解为 **“人才引进审批 + 居留许可管理”双线并行**；
- 获批后通常仍需进入澳门关于居留的法定程序，由出入境/居留管理机关按法律制度处理。

### 2.2 已能坐实的居留与续期要点

虽然本轮仍未直接抓到《施行细则》全文逐条，但有几条已经可以写实：

- 配套的 **第19/2023号行政法规《人才引进法律制度施行细则》** 明确涵盖 **“办理居留许可的手续和须附同的文件及资料，以及居留许可的有效期及续期程序”**；也就是说，人才引进获批后并不是停留在“入选名单”，而是会继续进入正式的 **居留许可办理 + 有效期 + 续期** 法定程序；
- 身份证明局公开的 **PS-1065 居民身份证续期申请** 页面已明确提到：持有非永久性居民身份证并 **获治安警察局发出的居留证明文件** 的人士，可据此办理后续证件手续；
- 同一页面还进一步点名：如属 **第7/2023号法律《人才引进法律制度》第3条（三）项的高级专业人才**，在申请由非永久性居民转为永久性居民时，须递交 **人才发展委员会发出的确认居留许可有效文件**；
- 这说明至少对 **高级专业人才** 而言，人才引进获批后形成的是一条可以落到 **非永久性居民身份证—居留许可持续有效—满七年后申请永久性居民相关审查** 的制度链条，而不是一句泛泛的“给身份”。

### 2.3 续期逻辑通常不会脱离“原来为何获批”

在上述已坐实信息基础上，更稳的判断是：

- 续期不会脱离申请人原先获批所依赖的基础；
- 也就是你是否仍在澳门从事相应工作、履行相应职责、维持原有专业或雇佣关系，或继续具备当初获批所依赖的条件；
- 如果原本是以特定雇主、岗位、项目、机构或专业身份获批，续期通常不会完全脱离这些基础事实。

但本轮仍然**不能写死**的地方也要明确：
- **人才引进居留许可的具体类别名称、首次批准期限、每次续期年限**，本稿尚未直接抓到《施行细则》或治安警察局办事页的稳定原文，因此暂不硬写数字；
- 目前能坐实的是：**有法定“有效期及续期程序”存在，且与治安警察局居留证明文件、人才发展委员会确认文件直接挂钩。**

也因此，澳门人才引进更适合：
- 已经有明确机构承接、岗位承接、项目承接的人；
- 愿意长期在澳门或“澳门 + 横琴”双城布局的人。

不太适合：
- 只想先拿身份、后续是否在澳发展并不确定的人；
- 没有稳定职业落点，只想用身份做备用选项的人。

### 2.3 永久性居民 / 长期身份，不建议在首稿里写死捷径

澳门关于永久性居民资格、通常居住、居留连续性等问题，本身就属于法律概念较强、个案差异较大的内容。

所以本稿先保持稳妥边界：

- **不把人才引进直接写成“几年后必转永久居民”**；
- 只提示：如果你关心长期身份，必须把 **居留许可类型、续期要求、通常居住认定、离澳时间影响、家庭成员安排** 一并看；
- 相关具体转化路径，建议后续继续补身份证明/出入境主管部门或官方问答口径后再落文。

## 3. 横琴联动、住房、税务、就业环境：澳门真正有吸引力和真正有门槛的地方

### 3.1 横琴联动，是澳门这类选择最大的增量

如果只看澳门本地岗位市场，很多人会觉得机会面偏窄。这是事实的一部分。

但澳门的现实优势，不完全在“澳门单城”，而在 **“澳门 + 横琴粤澳深度合作区”** 的联动：

- 澳门提供国际化、葡语国家联系、旅游会展品牌、法制与营商环境特色；
- 横琴提供更大的空间载体、产业园区、办公居住承接和与内地市场的连接；
- 对做中医药大健康、文旅会展、科技研发、跨境金融服务、跨境专业服务的人来说，这种联动比单看澳门本岛更有意义。

更适合怎么理解：
- 澳门像“品牌、制度和国际接口”；
- 横琴像“空间、项目落地和产业承载”。

### 3.1.1 本轮已能坐实的横琴澳门人才支持文件

这部分本轮先只写已抓到正式文件或官方办事页面的内容：

**1）就业支持：第5/2024号民生事务局规范性文件《横琴粤澳深度合作区支持澳门青年就业暂行办法》**
- 发布机关：横琴粤澳深度合作区民生事务局；
- 印发日期：**2024年12月31日**；
- 施行日期：**2025年2月1日**；
- 有效期：**三年**；
- 适用对象：在合作区就业的 **18—45周岁澳门特别行政区居民**，以及聘用澳门青年的合作区用人单位；
- 澳门青年基本条件包括：
  1. 具有 **专科及以上学历**，或职业资格证书，或可被认可的职业技能证书；
  2. **仅在合作区用人单位就业**，与同一单位签订 **一年及以上或无固定期限劳动合同**；
  3. 在合作区 **连续缴纳社保6个月或以上**；
  4. **实际工作地点、工资薪金所得税缴纳地均在合作区**；
  5. 无犯罪记录；
- 已坐实的支持标准：
  - **就业补贴**：每人每月 **4000元人民币**，最长 **36个月**；
  - **聘用奖励**：用人单位按每人每月 **1000元人民币**，最长 **36个月**；
  - **社会保险补贴**：按单位为澳门青年实际缴纳的城镇职工养老、医疗（生育）、失业、工伤保险中 **单位缴费部分全额补贴**，最长 **36个月**；
  - **职业技能提升补贴**：按广东相关政策标准 **配比30%一次性补贴**。

**2）澳门居民招聘专区与澳门青年服务页面**
- 横琴合作区民生事务官方页面已设置 **“澳门居民招聘专区”** 与 **“澳门青年服务”** 入口，说明合作区对澳门居民就业承接并非口号层面，而是已有公开服务承接端口。

这部分可以支持本文的一个更稳判断：
- 横琴对澳门居民至少已形成 **就业承接 + 用工奖励 + 社保补贴 + 职业技能提升** 的正式支持链条；
- 但**不是泛指所有澳门人才**都自动享受，当前最硬的正式文件之一，是偏向 **澳门青年就业** 的专项办法。

### 3.2 税务环境：澳门整体税负通常更友好，但不能简单等同“全部低税”

澳门对很多高收入专业人士、企业管理层、国际业务人才有吸引力，一个现实原因就是税务环境通常更友好，整体税制也相对简洁。

但更稳的写法是：
- 澳门整体税负环境通常优于很多内地城市；
- 如果你的收入、工作地点、纳税地点、合同主体、跨境任职关系牵涉到澳门与横琴/内地两边，实际税务处理不能只看一句“澳门低税”；
- 涉及 **个税归属、补贴、跨境工作安排、双城任职** 时，最好结合具体合同和税务规则看。

本轮已能坐实到的横琴税务相关点只有：
- 在第5/2024号民生事务局规范性文件《横琴粤澳深度合作区支持澳门青年就业暂行办法》中，澳门青年申请就业补贴的条件之一就是 **工资薪金所得税缴纳地均在合作区**；
- 这至少说明横琴对澳门青年就业支持已把 **纳税地在合作区** 作为明确条件之一，而不是只看劳动合同。

也就是说：
- 对纯在澳工作的人，澳门税务环境通常是加分项；
- 对跨境双城工作的人，税务优势可能存在，但需要结合横琴与大湾区相关政策具体判断；
- **横琴针对澳门高端/专业人才的个税优惠、财政补贴返还等正式文件，本轮尚未抓到稳定原文，继续留待复核。**

### 3.3 住房：澳门住房成本往往不是它的优势项

和内地不少抢人城市相比，澳门住房端更需要说实话：

- 澳门本地住房与租房成本，通常**不是**它吸引青年人才的核心优势；
- 至少在“普通本科毕业生刚起步”这个层面，澳门不太像珠海、杭州、武汉那种容易讲出普惠租房补贴故事的城市；
- 如果工作主要落在澳门、居住主要落在横琴或珠海，再利用口岸通勤，反而更接近很多现实选择。

因此更适合的判断是：
- 澳门的人才吸引力，主要不在“便宜住”；
- 而在 **制度身份机会、国际化平台、行业资源、横琴联动**。

本轮补充后的边界也更清楚：
- **横琴支持澳门青年就业暂行办法** 已坐实就业补贴、聘用奖励、社保补贴，但**没有直接写成住房补贴文件**；
- 因此本文仍**不把横琴住房支持写实为现行通用福利**，只保留“横琴具备更强空间承接能力”的判断；
- 澳门本地针对引进人才的公屋、人才住房或租住专项支持，本轮仍未抓到足够硬的正式口径。

### 3.4 就业环境：机会集中、上限取决于赛道匹配

澳门就业环境的特点很鲜明：

- 体量小，岗位池不像北上深广那样宽；
- 但在 **旅游休闲、酒店会展、现代金融、专业服务、中医药大健康、葡语国家经贸与法律语言服务、教育科研** 等方向，仍有其特殊位置；
- 真正适合的人，往往不是“海投所有岗位”，而是 **和澳门重点产业方向高度匹配的人**。

所以澳门并不是一个普适型就业目的地，而是一个 **强匹配型、强方向型** 的人才目的地。

## 4. 适合人群与不适合人群

### 4.1 更适合哪些人

**1）已经具备较强履历的中高端专业人才**
- 比如硕博、专业执照持有人、行业骨干、国际化人才；
- 他们更容易在“优秀人才 / 高级专业人才”层级里找到现实入口。

**2）能与澳门重点产业直接对接的人**
- 文旅会展、酒店管理、现代金融、专业服务、中医药大健康、高新技术、教育科研等；
- 尤其适合能服务澳门经济适度多元战略的人。

**3）接受澳门与横琴联动发展的人**
- 如果你愿意把工作、项目、办公、居住拆到“澳门 + 横琴”双节点看，澳门的可行性会明显提高。

**4）看重国际化环境、葡语国家连接、制度型机会的人**
- 澳门在这一点上有独特性，尤其适合做跨境、涉葡语国家、国际化服务的人。

### 4.2 不太适合哪些人

**1）只想走“学历落户捷径”的人**
- 澳门不是内地那种“学历够了、年龄合适、网上申报就基本能办”的典型城市。

**2）希望拿到统一普惠现金补贴的人**
- 澳门当前更应看制度引进和居留资格，而不是普惠生活补贴。

**3）职业方向与澳门重点产业关联弱的人**
- 如果你的行业与澳门重点发展方向匹配度不高，可落岗位会明显变少。

**4）不愿接受居留续期和在澳实际发展约束的人**
- 如果你只是想备份一个身份，但不准备在澳长期工作生活，澳门的人才引进制度未必适合你。

## 5. 速查表

| 维度 | 澳门当前更稳妥的判断 |
|---|---|
| 政策主线 | **不是内地式落户主线，而是法定人才引进制度 + 居留许可管理** |
| 人才类别 | **高端人才、优秀人才、高级专业人才** |
| 更看重什么 | **学历/资历/成果/行业地位/岗位承接/产业匹配度** |
| 是否适合普通应届生直接冲 | **通常不如内地强二线城市友好** |
| 居留逻辑 | **先看能否进入人才引进制度，再看居留许可与续期** |
| 续期核心 | **大概率仍围绕原获批基础：岗位、机构、项目、专业身份等** |
| 永久身份 | **不能在首稿里简单写成“几年必转永居”，需看法律与通常居住等条件** |
| 横琴联动 | **非常关键，是澳门吸引人才的重要增量** |
| 住房 | **通常不是澳门的核心优势项** |
| 税务 | **整体税务环境通常较友好，但跨境安排要看具体税务归属** |
| 就业环境 | **岗位池较小，但在文旅会展、金融、专业服务、中医药、高新技术等方向有特殊性** |
| 更适合的人 | **中高端专业人才、国际化人才、与澳门重点产业匹配的人** |
| 不太适合的人 | **只想拿普惠补贴、只想走学历落户捷径、产业匹配度弱的人** |

## 8. 待复核

### 已明确需要下一轮继续打实的点
1. **第7/2023号法律、第19/2023号行政法规、第20/2023号行政法规** 的《公报》全文链接，以及《施行细则》中关于居留许可有效期、续期程序、税务优惠申请审批的逐条原文；
2. **高端人才** 认定标准所列全部成就/职衔清单；
3. 除大健康产业外，**优秀人才 / 高级专业人才** 在现代金融、高新科技、文化体育等产业批次的正式申请条件、评分表、开放时间；
4. 人才引进获批后对应的 **居留许可类别名称、首次批准期限、续期年限、续期材料、家属安排**；
5. 是否存在官方明确的 **“通常居住 / 长期留澳 / 转永久性居民”** 问答或办事指南，可据此补充长期身份路径；
6. 横琴粤澳深度合作区对澳门引进人才在 **执业便利、住房支持、个税优惠、项目承接** 方面的现行正式文件；
7. 澳门本地是否存在针对引进人才的 **公屋/人才住房/租住支持** 的更明确官方口径；
8. 澳门重点产业的最新官方口径，是否已进一步收敛到 **综合旅游休闲、会展商贸、现代金融、中医药大健康、高新技术、文化体育** 等具体赛道。

### 当前写作边界
- 本稿**没有**把澳门写成“学历落户城市”；
- 本稿**没有**编造现金补贴、统一年龄线或统一社保月数；
- 本稿把未抓到稳定原文的细项，都明确留在“待复核”里，后续可继续精修。
