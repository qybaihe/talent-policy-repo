# 广州 人才引进政策速查

- 状态：已重写为“满足什么条件 → 可以获得什么优势”结构
- 优先来源：广州市人民政府、广州市人社局、广州市财政局等官方公开信息

## 一句话判断
广州的市级优势比较清晰：如果你满足**学历型或技能型引进人才入户条件**，可以直接拿到**广州户口**；如果你暂时还不落户，但属于优秀人才，可以通过**人才绿卡**获得购房、住房公积金、教育、安居等便利；如果你是符合条件的**境外人才**，还有机会拿到**个税超过15%部分的财政补贴**。

---

## 1. 满足什么条件 → 可以获得什么优势

### 1.1 如果你符合学历型 / 职称型引进人才入户条件

**你需要大致满足：**
- 在广州**就业或创业**。
- 符合以下典型条件之一：
  - **博士研究生学历、博士学位或高级职称**：**50周岁以下**；
  - **硕士研究生学历或硕士学位**：**45周岁以下**；
  - **国内普通高校全日制本科+学士学位**、或**国（境）外学士学位**、或**中级职称**：**40周岁以下**。
- 与广州单位建立劳动关系，或在广州注册企业并满足创业人员口径。
- 实操中需依法缴纳社保，但学历型条款本身**未统一写死“满X个月社保”**。

**你可以获得：**
- **直接申请广州入户**，这是广州人才政策最核心的优势。
- 符合条件的引进人才，**配偶、未成年子女可随迁**。
- 对于本硕博人才来说，广州的落户年龄上限相对直观，判断成本较低。

**但要注意：**
- 广州市级统一政策中，**没有明确写成所有学历人才都可领取统一现金补贴**。
- 如果你更关注补贴金额，往往要继续看黄埔、南沙等区级政策。

---

### 1.2 如果你符合技术技能人才入户条件

**你需要大致满足：**
- 从事广州引进技术技能人才职业目录内相关职业；
- 获得对应资格证书或考核认定；
- 在广州工作，并**参加社会保险满半年以上**；
- 年龄一般符合以下分档：
  - 专业技术人员职业资格：**40周岁以下**；
  - 高级技师：**45周岁以下**；
  - 技师：**40周岁以下**；
  - 高级工：**35周岁以下**；
  - 紧缺工种人员：**40周岁以下**。

**你可以获得：**
- 通过技能型通道**落户广州**。
- 这意味着广州并不只偏好学历人才，技能人才也能直接把职业资格转换成户籍优势。

**但要注意：**
- 技能人才更依赖职业目录、证书等级、社保半年记录等硬条件，操作时要按申报指南逐条核对。

---

### 1.3 如果你属于高层次、高技能、市场化创新创业人才

**你需要大致满足：**
- 属于经广州认定或审核确认的**高层次、高技能人才**；
- 或属于薪酬、投资等市场化评价下能力突出、贡献明显的**创新创业人才**；
- 或属于重大项目建设、重点扶持企业/项目中的中高级管理人员、技术骨干等。

**你可以获得：**
- 通过特殊人才通道**落户广州**；
- 对高层次、高技能人才而言，政策还允许**放宽年龄限制**；
- 如果你先办了人才绿卡，之后符合入户条件，还可以享受**落户绿色通道**。

---

### 1.4 如果你暂时不想落户，但想先享受人才便利

**你需要大致满足人才绿卡条件：**
- 属于**非广州市户籍**国内外优秀人才；
- 一般要求每年在广州**工作或创业超过6个月**（A卡主要条件）；
- 在广州有**合法住所**；
- 品行良好、守法合规。
- 常见可申领A卡的人群包括：
  - 经认定的高层次、高技能人才；
  - 具有博士后经历者；
  - 研究生学历并获博士学位者；
  - **全球前300名大学硕士及以上学位**；
  - **全球前200名大学本科并获学士以上学位**；
  - 重点产业、重点项目、总部企业中高级管理和骨干技术岗位人员；
  - 在穗个税达到标准的国（境）外人才等。

**你可以获得：**
- **购房便利**：非广州户籍内地居民可按广州户籍居民待遇购买自住房；国（境）外人员可按规定购买1套自住房。
- **安居便利**：可享受人才公寓、共有产权住房等安居保障待遇。
- **住房公积金、居住证、教育、车辆等综合便利**。
- 符合入户条件后，可享**落户绿色通道服务**。

**但要注意：**
- 人才绿卡**不等于直接落户**；
- B卡和A卡待遇不完全一致，住房安居待遇以A卡更完整。

---

### 1.5 如果你是符合条件的境外人才

**你需要大致满足：**
- 在广州市行政区域内工作；
- 属于**境外高端人才或紧缺人才**；
- 在纳税年度内在广州工作**累计满90天以上（不含90天）**；
- 在广州依法缴纳个税；
- 紧缺人才通常还要求个人所得税应纳税所得额达到**30万元以上**。

**你可以获得：**
- 对在广州缴纳的个人所得税中，**超过按15%税负测算部分**给予财政补贴；
- 该补贴**免征个人所得税**；
- 每人每个纳税年度补贴额**最高500万元**。

**这条政策的特点是：**
- 对国际化人才非常有吸引力；
- 广州在这方面的优势比很多内地城市更直接、金额也更高。

## 2. 速查表

| 条件维度 | 广州人才引进口径 |
|---|---|
| 学历 | 本科40岁以下、硕士45岁以下、博士50岁以下是典型入户门槛 |
| 年龄 | 学历型和技能型年龄规则较清晰；高层次人才可放宽 |
| 社保 | 学历型未统一写死月数；技术技能人才明确要求社保满半年 |
| 户籍 | 面向非广州户籍人员，审核通过后迁入广州户口 |
| 职称/技能 | 中级职称、高级工、技师、高级技师均可对应不同入户通道 |
| 留学/国际化 | 全球前200/300高校、本科硕士博士、境外人才个税补贴都较有吸引力 |
| 核心优势 | **落户广州**、人才绿卡综合便利、境外人才个税补贴 |
| 补贴特点 | 市级普惠现金补贴不强，但**绿卡待遇和境外人才补贴很有辨识度** |

---

## 3. 当前更适合怎么理解广州
- 如果你最看重**落户门槛清晰**，广州是很好判断的城市；
- 如果你是海外背景、国际化背景、重点产业骨干，广州的**人才绿卡 + 个税补贴**很有优势；
- 如果你期待的是“本科落户直接发一大笔市级现金补贴”，广州市级层面并不算最典型，更多看区级。
