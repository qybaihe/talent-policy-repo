# 深圳各区本科硕士人才补贴清单（区级速查附表）

- 适用城市：深圳
- 整理目的：把“本科/硕士补贴由各区制定实施”这件事，进一步拆成**区级速查表**，便于后续继续补硬证据。
- 证据策略：**优先官方来源**；当前环境下，深圳市政府 HTTPS 页面抓取存在 SSL 兼容异常，因此本表优先采用**已在主文中坐实的市级官方知识库口径**作为总入口证据；对尚未逐区打到正式文件原文的部分，统一标为**待复核**。

## 1. 已坐实的总规则

### 1.1 市级已经坐实的事情
- 深圳市级目前**没有统一公布本科、硕士固定补贴标准**；
- **本科、硕士“新引进优秀青年人才入户补贴”由各区制定和实施**；
- 公开查询路径已由深圳政府在线业务知识库明确：
  - **i深圳 App → 青年人才专区 → 生活服务 → 新引进优秀青年人才入户补贴**；
  - 或咨询**工作单位注册地所在辖区人力资源局**。

### 1.2 当前已被市级知识库公开点名的辖区
- 福田区
- 罗湖区
- 盐田区
- 南山区
- 宝安区
- 龙岗区
- 龙华区
- 坪山区
- 光明区
- 大鹏新区
- 深汕特别合作区

> 注：上面这个名单，至少可以坐实“这些区/新区属于当前公开咨询范围内的实施辖区”；但**并不当然等于每区当前都存在完全相同、同步更新、同额度的本科/硕士补贴政策**。

## 3. 如何正确理解这张表

### 3.1 当前能安全写实的边界
当前最稳妥、最不容易写错的结论只有三层：
1. **深圳市级统一稳定补贴是博士 10 万元生活补贴**；
2. **本科、硕士补贴不是全市统一标准，而是区级制定实施**；
3. **区级申请应以 i深圳 App 青年人才专区或工作单位注册地区人力资源局的当期口径为准**。

### 3.2 当前不能冒进写死的字段
在逐区正式文件或当期申报指南未逐一打实前，以下字段都不宜写死：
- 本科、硕士分别补多少；
- 是否都要求深圳户籍；
- 社保必须满几个月；
- 年龄是不是统一 30/35/40 岁；
- 是按劳动关系所在地、社保缴纳地还是单位注册地审查；
- 是否存在“年度预算用完即止”“阶段性暂停受理”或“仅限特定申报批次”。

### 3.3 本轮进一步坐实的共性判断
虽然还没逐区坐实到正式文件正文，但从深圳市政府知识库答复结构看，当前至少可以正式写实三点：
- **政策层级**：本科、硕士补贴是区级事项；
- **查询入口**：统一公开入口是 **i深圳 App 青年人才专区**；
- **辖区咨询口径**：以**工作单位注册地所在辖区人力资源局**为准。

因此做实务判断时，优先先问清三件事：
- 你是否已经取得深圳户籍；
- 你的单位注册地在哪个区；
- 你是否能在 i深圳 App 的青年人才专区看到当前区级申报事项。

### 3.4 本轮检索受限说明
- 当前环境下，`sz.gov.cn` 部分页面通过命令行抓取时存在 **SSL / 椭圆曲线握手异常**，导致无法稳定批量读取区级网页正文；
- 因此本轮对“各区金额、年龄、社保月数、是否仍执行”等字段，继续坚持 **未见区级正式原文 = 不写死**；
- 后续如可在浏览器环境或兼容 SSL 的抓取环境下逐区打开 i深圳 事项页、区人力资源局通知页，可继续把各区字段逐项打实。

## 4. 推荐补证顺序
如果后续继续深挖，建议按以下顺序打硬证据：
1. **南山、福田、宝安、龙岗、龙华**（申请量通常更大，参考价值最高）；
2. 再补 **坪山、光明、罗湖、盐田**；
3. 最后补 **大鹏新区、深汕特别合作区**。

优先寻找的官方材料类型：
- 区人力资源局/区政府官网的申报通知；
- 广东政务服务网分区事项页；
- i深圳 App 事项说明页截图或文案；
- 区级政策文件、实施细则、常见问答。
