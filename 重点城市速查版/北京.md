# 北京 人才引进政策速查

- 状态：已重写为“满足什么条件 → 可以获得什么优势”结构
- 优先来源：北京市人民政府、北京市人力资源和社会保障局、北京市人才工作局等官方公开信息

## 一句话判断
北京更像是“高门槛落户型政策”城市：如果你属于**高层次/紧缺急需人才**，或属于**符合条件的引进毕业生**，核心优势是**拿到北京户口**，并可带来配偶/未成年子女随迁；但**市级统一现金补贴、租房补贴、购房补贴并不突出**，更多要看区级和专项计划。

---

## 1. 满足什么条件 → 可以获得什么优势

### 1.1 如果你属于高层次 / 紧缺急需人才

**你需要大致满足：**
- 属于北京重点支持的人才类型之一，如：
  - 科技创新人才
  - 文化创意人才
  - 体育人才
  - 国际交往中心建设人才
  - 金融人才
  - 教育、科研、医疗卫生健康专业人才
  - 高技能人才
  - 自由职业者中的优秀人才
  - 优秀创新创业团队成员
  - 其他经综合评价、推荐或认定的特殊人才/紧缺急需人才
- 通常满足以下条件之一：
  - 国内外取得**硕士及以上学位**；
  - 具有**高级专业技术职称**；
  - 经人才引进综合评价合格；
  - 经各区或局级单位推荐；
  - 经相应人才认定委员会认定推荐；
  - 属于特殊特艺人才或其他紧缺急需人才。
- **年龄原则上不超过45周岁**；“三城一区”可放宽至**50周岁**；特别优秀者可进一步放宽。
- 一般需在北京聘用单位**工作满2年**。
- 用人单位需依法为你**缴纳社会保险**。
- **无刑事犯罪记录**。

**你可以获得：**
- **调京落户**资格，这是北京人才引进最核心的优势。
- **配偶和未成年子女可随调随迁**。
- 无自有住房时，可落在：
  - 用人单位**集体户**；
  - 人才公共服务机构**集体户**。
- 属于重大人才工程入选者、重要科技奖项获奖人等绿色通道对象的，**最快5个工作日**可完成引进手续。
- 如果你是创新创业团队核心成员，还可能通过创业业绩直接进入引进通道。

**但要注意：**
- 北京市级统一政策中，**没有特别明确的普惠型人才现金补贴**。
- **租房补贴、购房补贴、人才公寓**更多取决于区级政策，需单独核实。

---

### 1.2 如果你是符合条件的引进毕业生

**你需要大致满足：**
- 属于**非北京常住户口**应届毕业生，或**毕业两年内初次就业**毕业生。
- 列入国家统一招生计划，不属于定向、委培。
- 就读最高学历期间**未与任何单位建立劳动关系**。
- 按时取得学历学位。
- 学历条件通常满足以下之一：
  - **硕士、博士**毕业生；
  - 北京高校本科毕业生；
  - 京外“双一流”建设高校本科毕业生；
  - 文化、体育、教育、医疗等特定岗位对应本科毕业生。
- 年龄一般要求：
  - 本科**26周岁以下**；
  - 硕士**30周岁以下**；
  - 博士**35周岁以下**。
- 用人单位在北京注册、经营正常、依法纳税，并具备档案管理或委托档案服务能力。
- 用人单位应依法建立劳动关系并缴纳社保。

**你可以获得：**
- 通过单位申报后，获得**北京落户资格**。
- 可根据单位安排，选择：
  - **工作满半年**办理落户；
  - 或**工作满三年**办理落户。
- 郊区教师、社区卫生服务机构等部分岗位，虽落户时间较长，但仍可走引进通道。
- 无自有住房时，同样可通过**集体户**等方式落户。

**但要注意：**
- 这条通道的核心仍是**落户**，不是统一发钱。
- 市级政策中**没有明确统一的本科/硕士/博士安家补贴**。
- 区级园区、重点单位可能另有配套奖励，需单独复核。

---

### 1.3 如果你是创新创业人才 / 团队核心成员

**你需要大致满足：**
- 属于符合北京重点产业方向的创新创业团队；
- 团队近3年累计自主投入或股权融资达到较高规模，例如：
  - 自主投入**5000万元以上**或股权类现金融资**7000万元以上**，主要创始人和核心合伙人可申请；
  - 更高融资/投入规模下，团队成员也可在推荐下申请。
- 在校或休学创业毕业生，如本人为发起人/主要创始人、持股达到要求、企业符合北京重点产业且成效突出，也可能走引进毕业生通道。

**你可以获得：**
- 将创业成绩直接转化为**北京落户资格**。
- 团队核心成员也有机会一并纳入引进范围。
- 享受北京对创新创业人才的落户倾斜。

**但要注意：**
- 创业项目资金、创业补贴、办公场地支持等，更多属于专项项目或区级政策，**不能直接概括成全市统一普发待遇**。

## 2. 速查表

| 条件维度 | 北京人才引进核心口径 |
|---|---|
| 学历 | 高层次人才不完全按学历一刀切；引进毕业生常见为本科/硕士/博士 |
| 年龄 | 高层次人才原则45岁以下，部分可放宽到50岁；毕业生一般本科26/硕士30/博士35以下 |
| 社保 | 市级未统一写明“连续X个月”；但一般要求稳定劳动关系和依法缴社保 |
| 户籍 | 面向外埠人才和非京户毕业生；审核通过后办理北京户口 |
| 职称/技能 | 高级职称、高技能人才是重要通道 |
| 留学 | 海外取得硕士及以上学位者可纳入高层次/紧缺急需人才范围 |
| 核心优势 | **落户北京**、配偶和未成年子女可随迁 |
| 补贴特点 | **市级统一现金补贴不突出**，区级差异大 |

---

## 3. 当前更适合怎么理解北京
- 如果你最看重的是**北京户口**，北京很有吸引力；
- 如果你最看重的是**拿现金补贴、租房补贴、购房补贴**，北京并不是市级普惠最突出的城市；
- 如果你已进入重点单位、重点产业、重点科研或高端服务岗位，北京的优势会更明显，因为**平台资源 + 户口价值**通常大于直接补贴。

## 6. 待复核
- **已清除/已坐实**：
  - 高层次人才一般条件中的年龄上限、工作年限已由《北京市引进人才管理办法（试行）》原文坐实：**一般在聘用单位工作满2年，年龄原则上不超过45周岁，“三城一区”可放宽至50周岁**；
  - 引进毕业生年龄条件已由《北京市引进毕业生管理办法》原文坐实：**本科26周岁以下、硕士30周岁以下、博士35周岁以下**；
  - 引进毕业生办理节奏已由 2024 年官方解读进一步坐实：**可协商选择工作满半年或满三年办理落户**；
  - 市级文件层面**未见统一普惠型现金补贴**，可明确写为“核心优势是落户，不是市级统一发钱”。
- **仍待复核**：
  - 各区是否对引进人才/引进毕业生另设租房补贴、人才公寓、购房支持；
  - 当年度申报时对社保、个税、劳动合同、实际到岗时间的系统审查口径；
  - 自贸区、重点园区、重点企业是否存在更细化的专项落户便利条件。
