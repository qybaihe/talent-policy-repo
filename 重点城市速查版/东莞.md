# 东莞 人才引进政策速查

- 状态：新增首轮结构化稿；本轮优先用**东莞市政府正式文件**搭主线，把“已坐实”和“待复核”分开写，避免把宣传口径误写成现行待遇
- 优先来源：东莞市人民政府、东莞市人民政府办公室等官方公开信息

## 一句话判断
东莞这座城市现在最值得看的，不只是“能不能落户”，而是**入户主线已经明显放宽**：一条是**人才入户**，适合学历、职称、技能和企业骨干；另一条是 2024 年起更强势的**户籍改革主线**，包括**经常居住地登记户口**、**大专及以上学历毕业生“先入户后就业”**、以及**放宽在职人才入户年龄限制**。如果你更关心安居，东莞市级已经能坐实：**新入户人才可申领最长 3 年、每年 6000 元租房补贴**，并可叠加**市属住房租赁企业房源最长 3 年、每年 6000 元租金优惠**；如果你属于**特色人才**，住房、购房、生活补助力度会明显上一个台阶。城市空间上，东莞官方已明确把**松山湖、滨海湾新区**列为人才住房重点布局区域。

---

## 1. 满足什么条件 → 可以获得什么优势

### 1.1 如果你符合东莞人才入户基本条件

**你需要大致满足：**
- 未参加国家禁止的组织及活动；
- 在东莞**依法参加社会基本养老保险**；
- 并符合东莞市《人才入户实施办法》规定的准入条件之一。

**典型可走的人才入户类型包括：**
1. **学历型**
   - 国（境）外学习并获得**硕士以上学位**：**50周岁以下**；
   - 国（境）外学习并获得**学士学位**：**45周岁以下**；
   - 国内普通高等教育**全日制硕士及以上**：**50周岁以下**；
   - 国内普通高等教育**全日制本科**：**45周岁以下**；
   - 国内普通高等教育**全日制大专**：**40周岁以下**；
   - **非全日制本科以上学历**：在东莞连续缴纳养老保险满 **3 年**，且**35 周岁以下**。
2. **职称型**
   - **高级职称**：**50周岁以下**；
   - **中级职称**：**45周岁以下**；
   - **初级职称**：**40周岁以下**；
   - 同时应具备**中技或中等教育以上学历**。
3. **技能型**
   - **高级技师**：**45周岁以下**；
   - **技师**：**40周岁以下**；
   - **高级工**：**35周岁以下**；
   - **中级工**：在东莞连续缴纳养老保险满 **3 年**，且**30 周岁以下**；
   - 职业资格需与岗位匹配，且工种符合东莞**紧缺急需职业（工种）目录**。
4. **其他人才类型**
   - 省内职业学校、技工院校学制教育毕业两年内人员；
   - 世界技能大赛获奖者、中华技能大奖、全国/广东省/东莞市技术能手、东莞市“首席技师”“莞邑工匠”等表彰类技能人才；
   - **企业自评人才**：由园区、镇街制定年度计划，企业可推荐急需骨干申请入户。

**你可以获得：**
- 通过**人才入户**迁入东莞户籍；
- 配偶、未成年子女可按规定随迁；
- 对很多制造业、研发和技能岗位人才来说，东莞的人才入户不只认学历，也给了**技能和企业骨干通道**。

**但要注意：**
- 2020 年《人才入户实施办法》是当前人才入户基础文件，但 2024 年户籍改革后，实操时要结合最新改革口径一起看；
- 企业自评人才通道依赖**各园区、镇（街道）年度计划**，不是全市统一静态名额。

### 1.2 如果你更适合走 2024 年起的户籍改革主线

**这条线目前已能从官方文件坐实三件事：**

#### （1）经常居住地登记户口制度
**你需要大致满足：**
- 非东莞户籍；
- 在东莞**合法稳定居住就业**；
- 在用人单位正常参加东莞社会养老保险；
- 持有东莞有效《广东省居住证》**连续满 1 年**。

**你可以获得：**
- 直接申请入户东莞；
- 配偶及未成年子女可以随迁；
- 入户地址顺序为：
  1. 本人或直系亲属自有产权住宅；
  2. 工作单位集体户；
  3. 居住证登记地址所属园区、镇（街道）**新型社区集体户**。

#### （2）大专及以上学历毕业生“先入户后就业”
**你需要大致满足：**
- 非东莞户籍；
- **全日制大专及以上学历毕业生**；
- **毕业 5 年内**；
- 已在东莞申报居住登记。

**你可以获得：**
- 按“**先入户后就业**”原则先落东莞户口，再找工作；
- 如本人或直系亲属无自有产权住宅，可落入东莞专门设立的**人才公共户**。

#### （3）放宽在职人才入户年龄限制
**目前已坐实的最新上限为：**
- **全日制硕士及以上 / 高级职称 / 技师及以上技能人才 / 表彰类技能人才**：放宽至 **55周岁以下**；
- **全日制本科 / 中级职称**：放宽至 **50周岁以下**；
- **全日制大专 / 初级职称**：放宽至 **45周岁以下**；
- **高级工**：放宽至 **40周岁以下**；
- **中级工**：放宽至 **35周岁以下**。

**这条主线的意义是：**
- 东莞现在不只是“人才认定后才能落户”，而是明显把入户门槛往**长期居住人口、青年毕业生、在岗人才**方向打开；
- 对想先拿户口再找机会的人，大专及以上毕业生“先入户后就业”很有辨识度；
- 对制造业城市来说，放宽技能和在职人才年龄限制，非常实用。

### 1.3 如果你最在意安居：新入户人才和新就业人才能拿到什么

**目前市级已能稳稳坐实：**

#### 新就业人才
**你需要大致满足：**
- 符合东莞住房保障条件；
- 属于东莞《人才安居办法（试行）》口径下的新就业人才。

**你可以获得：**
- 申请入住**公共租赁住房**；
- 或享受市属住房租赁企业房源**最长 3 年、每年 6000 元的租金优惠**。

#### 新入户人才
**你需要大致满足：**
- 符合东莞住房保障条件；
- 属于东莞《人才安居办法（试行）》口径下的新入户人才。

**你可以获得：**
- 申请入住**公共租赁住房**；
- 或领取**6000 元/年租房补贴**，保障期限**最长不超过 3 年**；
- 如果选择租房补贴且入住市属住房租赁企业房源，**还可同时享受最长 3 年、每年 6000 元租金优惠**。

**这一条怎么理解最有用：**
- 东莞市级对普通新入户 / 新就业人才的住房支持，已经不是“完全没有”，而是有一条很清晰的**过渡性安居线**；
- 金额不是珠三角最激进，但政策结构非常明确：**公租房 / 租房补贴 / 租金优惠三选一或组合承接**；
- 对刚落户、刚就业的人，这条线比模糊宣传更实在。

---

### 1.4 如果你属于创新人才、高技能人才或研发人才

**你需要大致满足：**
- 属于东莞《人才安居办法（试行）》纳入的人才范围；
- 常见口径包括：
  - **创新人才**；
  - **高技能人才**；
  - **研发人才**；
  - 或具有全日制本科及以上学历、教育部认可的境外学历；
  - 或具有大专学历 / 中级技工及以上职业资格并被列入紧缺人才；
  - 或符合条件在东莞创业的人才。

**你可以获得：**
- 申请租购**人才住房**；
- 创新人才可同步申请**综合补贴**：
  - 正高级职称：最高 **30 万元**，分 5 年等额发放；
  - 副高级职称或博士学位：最高 **20 万元**，分 5 年等额发放；
  - 硕士学位：最高 **6 万元**，分 3 年等额发放；
  - 中级职称或符合个税条件的限定用人单位人才：最高 **6 万元**，分 3 年等额发放；
  - 初级职称或本科学历、且在限定用人单位工作的创新人才：由镇街（园区）制定，**不低于 1 万元/人**，其中市级“倍增计划”试点企业创新人才**不低于 2 万元/人**。
- 高技能人才可同步申请**紧缺急需高技能人才引进综合补贴**以及租购人才住房。

**但要注意：**
- 这些补贴不是“所有本科 / 硕士入户人才普发”，而是跟**创新人才 / 高技能人才 / 研发人才身份**和**限定用人单位**绑定；
- 具体申报名单、企业范围、年度执行通知仍要看主管部门和镇街园区执行口径。

### 1.5 如果你属于东莞特色人才

**这是东莞补贴力度最强的一层，但门槛也明显更高。**

**你需要大致满足：**
- 经认定、评定为东莞**特色人才**。

**你可以获得的住房保障已能坐实：**
1. **租住人才公寓**：优先提供租住优惠；
2. **租房补贴**（未购房、未享受人才租住优惠或保障房租住优惠）：
   - 特级人才：最高 **6000 元/月**；
   - 一类人才：最高 **5000 元/月**；
   - 二类人才：最高 **4000 元/月**；
   - 三类人才：最高 **3000 元/月**；
   - 四类人才：最高 **2000 元/月**；
   - 补贴期限**不超过 3 年**；
3. **购房补贴**（首次认定前后一定时限内在莞首次购买首套商品住房）：
   - 特级人才：最高 **1000 万元**；
   - 一类人才：最高 **600 万元**；
   - 二类人才：最高 **400 万元**；
   - 三类人才：最高 **200 万元**；
   - 四类人才：最高 **100 万元**；
   - 从连续工作满 12 个月起可申请，**按 5 年等额发放**；
4. **公积金贷款优惠**：住房公积金贷款额度可按规定**上浮 20%**。

**生活保障已能坐实到什么程度：**
- 获认定、评定为特色人才后在东莞连续工作满 12 个月，可申领**5 年逐年递增的生活补助**；
- 目前本轮已能从原文稳定打实：
  - 特级人才：**第一年 5 万元、第二年 6 万元、第三年 7 万元、第四年 8 万元、第五年 9 万元**；
  - 一类人才：**第一年 4 万元、第二年 5 万元、第三年 6 万元、第四年 7 万元、第五年 8 万元**；
- 二类至四类生活补助标准，当前检索片段未完整打全，**暂列待复核**，不在本稿写死。

**其他综合服务已能坐实：**
- 子女入学可在工作单位或落户所在园区、镇（街道）选择，由教育部门安排至义务教育阶段公办或购买服务民办学校；
- 特色人才可获东莞**优才卡**，享受政务、商事、交通、就业、配偶安置、科研、培训等综合服务；
- 医疗上可享受二级及以上医院绿色通道，并按人才类别享受健康体检服务。

**这一层怎么理解：**
- 东莞真正“大额安居和生活补贴”的重心，不在普通入户人才，而在**特色人才**；
- 所以如果求职者只是普通本科 / 硕士 / 大专入户，不宜把特色人才补贴当作普惠口径；
- 但对于高层次创新人才，这套政策的住房和购房支持强度是非常能打的。

---

### 1.6 如果你关心松山湖、滨海湾等重点平台

**目前已能从官方文件直接坐实：**
- 东莞《人才安居办法（试行）》明确要求人才住房规划布局要与区域发展、产业发展、轨道交通等规划衔接；
- 并明确提出：**重点加强松山湖、滨海湾新区等广深科技创新走廊核心创新平台，以及松山湖科学城、水乡新城等创新节点的人才住房项目建设。**

**这意味着什么：**
- 松山湖、滨海湾不是“市场猜测中的重点区”，而是已经进入东莞市级人才住房布局文件；
- 对研发人才、科技人才、平台型企业骨干来说，这两个区域后续承接人才住房、项目配建、安居资源的确定性更高；
- 但到“具体谁能申请、房源多少、补贴多少”这一层，仍要继续看各平台的现行申报通知和项目清单。

**当前更稳妥的写法是：**
- **松山湖**：已被市级文件点名为人才住房重点建设区域，适合继续重点复核科研平台、高校、重点企业的人才住房和配套政策；
- **滨海湾新区**：已被市级文件点名为广深科技创新走廊核心平台的人才住房重点建设区域，适合继续重点复核新区产业人才、港澳青年、创新创业载体的安居支持；
- 本轮未直接抓到两地现行住房补贴正式文件全文，因此**不写死区级金额**。

## 2. 速查表

| 条件维度 | 东莞当前更稳妥的官方口径 |
|---|---|
| 学历 | 全日制大专、本科、硕士、博士都有明确人才入户或毕业生入户通道 |
| 年龄 | 2024 年后明显放宽：硕士/高级职称/技师等可放宽至 **55 岁以下** |
| 社保 | 人才入户通常要求在莞参加养老保险；部分非全日制/中级工条件明确要求 **连续 3 年** |
| 户籍 | 核心优势非常明确：**能落东莞户口** |
| 职称/技能 | 中高级职称、技师、高级工、中级工、表彰类技能人才均可作为通道 |
| 留学 | 境外学士、硕士以上学位均可按相应年龄条件走人才入户 |
| 市级安居 | **新入户人才 6000 元/年租房补贴，最长 3 年**；新就业人才可享市属租赁房源租金优惠 |
| 大额补贴重心 | 主要集中在**特色人才**、创新人才、高技能人才等分层体系 |
| 平台布局 | **松山湖、滨海湾新区**已被市级文件明确列为人才住房重点区域 |
| 最重要提醒 | 东莞要分清：**普惠入户线** 和 **高层次/特色人才补贴线**，不能混写 |

## 3. 当前更适合怎么理解东莞

- 如果你最看重的是**落户通道更宽**，东莞这两年是明显变强的；
- 如果你是**全日制大专及以上毕业生**，东莞的“**先入户后就业**”非常有辨识度；
- 如果你是制造业、技能型、研发型人才，东莞不是只看学历，而是给了较完整的**技能 + 企业骨干 + 创新人才**通道；
- 如果你刚落户、刚就业，东莞市级已经给出一条比较明确的**过渡性安居线**；
- 如果你真正关心“大额购房补贴、每月高额租补、生活补助”，那其实看的不是普通入户，而是**特色人才体系**；
- 空间上，东莞不能只看“全市平均”，**松山湖、滨海湾新区**这些平台更值得继续盯。

## 6. 已坐实 / 待复核

### 6.1 已坐实
- 东莞 2020 年《人才入户实施办法》已明确学历、职称、技能、企业自评等多类人才入户通道；
- 2024 年《深化户籍制度改革的实施意见》已坐实：
  - **经常居住地登记户口制度**；
  - **全日制大专及以上学历毕业生“先入户后就业”**；
  - **在职人才入户年龄限制进一步放宽**；
- 市级安居线已坐实：
  - **新就业人才**：可申请公租房，或享受市属住房租赁企业房源最长 3 年、每年 6000 元租金优惠；
  - **新入户人才**：可申请公租房，或享受 **6000 元/年租房补贴**，最长 3 年；如入住市属住房租赁企业房源，还可叠加最长 3 年、每年 6000 元租金优惠；
